# PSI to Bar Converter

Free online tool to convert PSI to Bar and Bar to PSI instantly.

🔗 **Live Site: [psibar.co](https://psibar.co)**

## Features
- Instant real-time conversion
- One-click swap between PSI↔Bar
- Copy button for quick sharing
- Conversion chart (1-3000 PSI)
- Common tire pressure values (30-42 PSI)
- Mobile-friendly design

## Use Cases
- Tire pressure conversion for cars and bicycles
- HVAC technician pressure readings
- Scuba diving tank pressure
- Industrial hydraulic systems

## Tech
Pure HTML/CSS/JS, deployed on Vercel.
